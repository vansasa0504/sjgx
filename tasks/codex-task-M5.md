## M5 阶段任务

### 阶段目标
端到端集成测试、性能压测达标、安全测试通过、国产化环境兼容验证、同城双活部署配置。

### 前置条件
M4 验收通过（全部 9 大模块后端 + 前端已就绪）。

### 本阶段交付物
- 端到端自动化测试套件（全链路：合作方注册→接入→校验→服务→消费→审计→计费）。
- 性能压测脚本（JMeter）与调优（连接池、缓存、SQL、JVM）。
- 安全加固：SQL 注入/XSS/CSRF 防护验证、接口防刷、漏洞扫描修复。
- 国产化适配验证：达梦 DM8 + OceanBase 实际环境测试、麒麟/UOS 兼容、X86/ARM。
- 同城双活部署配置（k8s/dev/ 双机房模拟）。

### 实现要求
1. 端到端测试用 Testcontainers + 真实协议 Mock，覆盖主链路与关键异常分支（接口不可达、格式错误、配额超限、熔断、断点续传、质量拦截、鉴权失败）。
2. 性能调优：HikariCP 连接池参数、Redis 连接池、JVM 堆/GC 参数、慢 SQL 优化、索引补充、缓存策略调优。
3. 安全加固：MyBatis-Plus 参数化查询防 SQL 注入复核、输入过滤防 XSS、CSRF Token、接口限流防刷（Sentinel）、敏感数据日志脱敏复查。
4. 国产化：切换达梦/OceanBase 实际驱动跑全部集成测试，修复方言不兼容 SQL（如分页、日期函数、JSON 函数差异）。
5. 双活：Redis Sentinel/Cluster、数据库主从、Kafka 多机房复制配置；故障切换演练脚本。k8s/dev/ 双机房部署描述。
6. 性能指标对照 NFR-P01~P07 逐项验证并记录数据：
   - 标准接口 P50≤200ms / P95≤500ms / P99≤1s
   - 定制接口 P50≤500ms / P95≤1s
   - ≥1000TPS（峰值 ≥2000TPS）
   - 单批次 100 万条 ≥100MB/s 断点续传
   - 单节点 ≥1 万条/秒接入、日均 ≥10 亿条
   - 加工日均 ≥5 亿条、延迟 ≤5 分钟
   - 缓存命中率 ≥90%、查询 ≤10ms、容量 ≥10TB（设计可扩展）
   - 千万级查询 ≤2s、亿级 ≤5s
7. 输出性能压测报告数据（每项指标的实测值与是否达标）。

### 测试要求
- 端到端全链路测试通过。
- 性能压测达标（NFR-P01~P07 全部，输出实测数据）。
- 安全测试：OWASP ZAP 扫描无高危，手工渗透 SQL注入/XSS/CSRF/越权/签名绕过/重放通过。
- 国产化：达梦 + OceanBase 全部集成测试通过。
- 双活：故障切换演练通过（单节点切换 ≤30s，集群恢复 ≤5min，数据零丢失，RPO≤5min/RTO≤30min）。

### 完成判定（本阶段验收标准）
- 全部性能指标达标（输出压测报告数据）。
- 安全扫描无高危漏洞。
- 达梦 + OceanBase 双环境测试通过。
- 端到端全链路自动化测试通过。
- 双活故障切换演练达标。
