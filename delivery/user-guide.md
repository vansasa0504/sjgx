# 用户操作手册

版本：0.1.0  
日期：2026-06-26  
维护者：Codex

## 角色

| 角色 | 权限范围 |
|---|---|
| 平台管理员 | 全模块配置、审批、监控、审计 |
| 合作方管理员 | 合作方资料、接口配置、质量反馈 |
| 消费方 | 数据目录浏览、申请、服务调用 |
| 运维人员 | 监控、日志、故障处理、巡检 |
| 开发人员 | 接入适配、服务配置、规则扩展 |

## 登录

通过 `platform-ui` 登录页进入系统，后端由 `platform-auth` 颁发 Token。菜单按角色权限动态展示。

## 合作方管理

1. 新建合作方档案。
2. 提交资质审核。
3. 配置接口、凭证和调用频次。
4. 完成准入、评级或退出。

字段说明：合作方名称、数据类型、行业属性、合规等级、服务质量阈值、接口地址、凭证密文、调用频次。

## 数据接入

1. 创建接入任务，选择协议 HTTP/WebService/SFTP/Kafka/MQ/DB/API-GW。
2. 选择格式 JSON/XML/CSV/Excel。
3. 配置字段映射和质量规则。
4. 测试验证，通过后上线。

导航路径：数据接入 → 接入任务 → 新建任务。测试失败时查看错误详情，常见原因包括协议不可达、格式解析失败、质量规则不通过。

## 数据服务

1. 注册服务并填写路由。
2. 完成定义、测试、发布。
3. 查看服务调用日志和错误率。
4. 需要停止时执行下线归档。

调用方需使用 API Key、HMAC 签名、时间戳和 nonce。连续重复 nonce 或过期时间戳会被拒绝。

## 数据目录

按主题、合作方、类型、场景浏览数据资产。授权用户可查看样本、统计信息和质量报告，并提交使用申请。

导航路径：数据目录 → 目录浏览 / 检索 / 申请使用。无权限数据只能查看元信息，不展示样本明细。

## 消费方管理

1. 注册消费方。
2. 审批权限。
3. 配置频次、数据量和服务范围配额。
4. 查看调用行为审计和违规预警。

## 数据质量

配置六维规则，查看校验结果、问题工单、整改状态、质量报告和评分评级。

## 计费统计

配置计费规则，生成日/月/季度账单，处理确认、异议、调整和结算。统计监管模块可查看调用量、传输量、成功率、缓存命中率和监管报表。

## 系统管理和大屏

管理员维护用户、角色、权限和配置。监控大屏展示运行、服务、合规、成本核心指标。
