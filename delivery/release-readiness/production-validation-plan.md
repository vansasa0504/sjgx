# 生产等价环境补测计划

> 所有卡片初始状态均为 `BLOCKED`。只有取得目标环境、书面授权、变更窗口、数据方案和回退批准后方可执行。本阶段没有执行以下任何测试。

## 1. 通用执行与回填规则

- 执行前冻结候选 commit、应用/中间件/数据库版本、拓扑、资源规格、数据规模与分布；敏感配置仅在受控系统保存。
- 原始输出、监控、日志、审批、计时和复核记录统一归档至 `CONTROLLED-LOCATION-TBD`，仓库只记录材料编号和脱敏摘要。
- 每卡均须记录开始/结束时间、执行角色、独立复核角色；阈值未达或数据校验失败记 `FAIL`，前置缺失记 `BLOCKED`，不得用 DEV 结果替代。
